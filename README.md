# Bazi static website

This is a plain HTML, CSS, and JavaScript website. It requires no npm installation and no build step.

## GitHub Pages deployment

1. Upload all files in this folder directly to the root of the GitHub repository.
2. In GitHub, open Settings > Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Select the `main` branch and `/ (root)` folder.
5. Save and wait for GitHub Pages to publish.

The repository root must contain `index.html`.
